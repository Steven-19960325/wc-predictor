    三源数据流. 每小时：API-Football(fixtures+standings)→Poisson xG 模型 + 真太阳时校正→可选调奇门服务起局。每2小时：The Odds API 拉 h2h 赔率，去水位后得市场隐含概率（500/月配额下的安全频率）。三者写入 predictions.json，本页同源读取、每小时自刷。
    关于权重. 模型与市场是有实证基础的信号，默认各 0.5；奇门为玄学参考、无统计验证效力，默认权重 0——它照常起局、照常显示真太阳时与盘面倾向，但不参与计算，除非你主动拨高。这样核心预测始终干净，玄学只作旁参。
    注意. 分析与学习用途；足球高方差，模型给的是概率分布而非定论，不构成投注建议。
